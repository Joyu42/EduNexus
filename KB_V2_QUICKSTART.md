# 知识库 V2 快速开始指南

## 🚀 快速开始

### 1. 启动应用

```bash
cd apps/web
npm run dev
```

访问: `http://localhost:3000/kb-v2`

### 2. 创建第一个文档

1. 点击左下角 **"新建文档"** 按钮
2. 输入文档标题，例如："我的第一篇笔记"
3. 按 Enter 或点击"创建"

### 3. 开始编辑

#### 基础格式化
- **粗体**: 选中文字，点击工具栏 **B** 按钮
- **斜体**: 选中文字，点击工具栏 *I* 按钮
- **标题**: 点击 H1、H2、H3 按钮

#### 插入内容
- **列表**: 点击列表按钮
- **链接**: 点击链接按钮，输入 URL
- **图片**: 点击图片按钮，输入图片 URL
- **表格**: 点击表格按钮

#### 快捷键
- `Ctrl+B`: 粗体
- `Ctrl+I`: 斜体
- `Ctrl+Z`: 撤销
- `Ctrl+Shift+Z`: 重做

### 4. 使用 AI 功能

#### AI 摘要
1. 编辑一些内容
2. 点击右侧 **"AI"** 标签页
3. 点击 **"生成摘要"**
4. 等待 AI 生成摘要

#### AI 思维导图
1. 在 AI 标签页中
2. 点击 **"生成思维导图"**
3. 查看可视化的文档结构
4. 可以全屏查看或导出

#### AI 问答
1. 在 AI 标签页底部
2. 输入问题，例如："这篇文档的主要内容是什么？"
3. 按 Enter 发送
4. AI 会基于文档内容回答

### 5. 查看文档大纲

1. 点击右侧 **"大纲"** 标签页
2. 查看自动提取的标题结构
3. 点击任意标题快速跳转

### 6. 搜索文档

1. 在左侧搜索框输入关键词
2. 实时过滤文档列表
3. 支持搜索标题、内容和标签

### 7. 导出文档

1. 选择要导出的文档
2. 点击右侧 **"属性"** 标签页
3. 点击 **"导出为 Markdown"**
4. 文件会自动下载

## 💡 使用技巧

### 自动保存
- 编辑器会在停止输入 2 秒后自动保存
- 右上角显示保存状态
- 无需手动保存

### 快捷键
- `Ctrl+B`: 切换左侧边栏
- `Ctrl+\`: 切换右侧面板
- `Ctrl+N`: 新建文档（预留）
- `Ctrl+K`: 搜索（预留）

### 文档组织
- 使用标签分类文档
- 利用搜索快速查找
- 查看"最近访问"快速回到常用文档

### AI 最佳实践
- 文档内容越详细，AI 生成的摘要越准确
- 使用清晰的标题结构，思维导图效果更好
- AI 问答基于文档内容，确保相关信息在文档中

## 🎯 常见场景

### 场景 1: 课堂笔记
1. 创建文档，标题为课程名称
2. 使用标题组织章节
3. 添加标签：课程代码、学期
4. 课后使用 AI 生成摘要复习

### 场景 2: 项目文档
1. 创建项目文档
2. 使用任务列表跟踪进度
3. 插入图片和链接
4. 使用思维导图查看项目结构

### 场景 3: 学习笔记
1. 记录学习内容
2. 使用 AI 问答测试理解
3. 定期导出备份
4. 通过搜索快速查找知识点

## ⚠️ 注意事项

1. **数据安全**: 数据存储在浏览器本地，定期导出重要文档
2. **浏览器兼容**: 推荐使用 Chrome/Edge 最新版
3. **AI 功能**: 需要网络连接和 API 配置
4. **性能**: 单个文档建议不超过 10000 字

## 🆘 常见问题

### Q: 文档没有自动保存？
A: 检查右上角保存状态，确保编辑器有焦点。

### Q: AI 功能不可用？
A: 检查网络连接和 API 配置，查看控制台错误信息。

### Q: 如何备份数据？
A: 使用"导出为 Markdown"功能逐个导出文档。

### Q: 可以导入已有的 Markdown 文件吗？
A: 目前支持复制粘贴内容，批量导入功能开发中。
