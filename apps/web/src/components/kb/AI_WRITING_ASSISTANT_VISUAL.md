## 动画效果详解

### 1. 淡入动画

```css
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

应用：
- 组件打开时
- 消息出现时
- 工具栏显示时

持续时间：200ms
缓动函数：ease-in-out
```

### 2. 滑入动画

```css
@keyframes slideInFromBottom {
  from {
    opacity: 0;
    transform: translateY(16px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

应用：
- 浮动按钮出现
- 侧边栏打开
- 消息添加

持续时间：200ms
缓动函数：ease-out
```

### 3. 缩放动画

```css
@keyframes scale {
  from { transform: scale(0.95); }
  to { transform: scale(1); }
}

应用：
- 按钮悬停
- 卡片悬停
- 图标交互

持续时间：200ms
缓动函数：ease-in-out
```

### 4. 打字机效果

```
实现方式：
1. 流式接收文本
2. 逐字添加到内容
3. 显示闪烁光标

视觉效果：
- 文字逐个出现
- 光标在末尾闪烁
- 平滑滚动跟随

光标样式：
- 宽度：2px
- 高度：16px
- 颜色：purple-500
- 动画：animate-pulse
```

### 5. 加载动画

```
三个跳动的点：

●  ●  ●  →  ●  ●  ●  →  ●  ●  ●
            ↑              ↑  ↑

实现：
- 三个圆点
- 依次跳动
- 延迟：0s, 0.2s, 0.4s
- 循环播放

样式：
- 大小：8px
- 颜色：purple-500
- 动画：animate-bounce
```

### 6. 脉冲动画

```css
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}

应用：
- 空状态图标
- 加载状态
- 等待反馈

持续时间：2s
循环：infinite
```
