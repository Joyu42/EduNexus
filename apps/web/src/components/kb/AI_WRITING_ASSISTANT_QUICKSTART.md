# AI 写作助手 - 快速开始

## 5 分钟快速上手

### 1. 基本使用

#### 打开助手

**方式一：点击浮动按钮**
- 在知识库页面右下角，点击紫色渐变的圆形按钮
- 按钮上有 ✨ 图标

**方式二：使用快捷键**
- Windows/Linux: `Ctrl + J`
- Mac: `Cmd + J`

#### 快速操作

打开助手后，你会看到 10 个彩色的快速操作按钮：

- **✨ 续写** - 让 AI 根据上下文继续写作
- **📝 改写** - 用不同的方式表达相同的内容
- **💡 润色** - 提升文字的质量和流畅度
- **✅ 修正** - 检查并修正语法和拼写错误
- **📊 总结** - 提取文档的核心要点
- **🔍 扩展** - 增加更多细节和例子
- **📋 大纲** - 生成结构化的内容大纲
- **💭 头脑风暴** - 生成创意想法
- **❓ 解释** - 通俗易懂地解释概念
- **🌐 翻译** - 翻译成其他语言

**使用方法**：
1. 点击任意快速操作按钮
2. AI 会立即开始生成内容
3. 内容以打字机效果实时显示

### 2. 选中文本操作

#### 使用浮动工具栏

1. 在编辑器中选中一段文字
2. 浮动工具栏会自动出现在选中文字上方
3. 点击工具栏上的操作按钮
4. AI 会针对选中的文字执行操作

**浮动工具栏包含**：
- 续写、改写、润色
- 翻译、总结、扩展

### 3. 对话交互

#### 自由提问

在输入框中输入你的需求，例如：

```
帮我写一段关于人工智能的介绍
```

```
这段话怎么改得更专业一些？
```

```
解释一下什么是双链笔记
```

#### 多轮对话

AI 会记住对话历史，你可以继续追问：

```
用户: 帮我写一段关于 AI 的介绍
AI: [生成内容]

用户: 再加上一些具体的应用场景
AI: [补充内容]

用户: 翻译成英文
AI: [翻译结果]
```

### 4. 使用生成的内容

每条 AI 回复下方都有操作按钮：

- **📋 复制** - 复制内容到剪贴板
- **🔄 重新生成** - 重新生成这条回复
- **✅ 插入** - 插入到文档光标位置
- **🔁 替换** - 替换选中的文字

### 5. 使用模板

#### 切换到全屏模式

1. 点击助手头部的 **全屏** 按钮（□ 图标）
2. 进入全屏工作模式

#### 选择模板

1. 点击头部的 **模板** 按钮
2. 浏览 12+ 种专业模板
3. 点击任意模板
4. AI 会根据模板生成内容

**模板分类**：

**写作辅助**：
- 会议纪要
- 邮件草稿
- 报告模板
- 学习笔记

**内容生成**：
- 文章大纲
- 博客文章
- 教程指南
- 案例分析

**学习辅助**：
- 概念解释
- 对比分析
- 问答生成
- 要点总结

## 常见场景

### 场景 1：写作卡壳

**问题**：不知道接下来写什么

**解决**：
1. 选中已写的最后一段
2. 点击浮动工具栏的 **续写**
3. AI 会根据上下文继续写作

### 场景 2：表达不够专业

**问题**：写的内容太口语化

**解决**：
1. 选中需要改进的段落
2. 点击 **改写** 或 **润色**
3. 在输入框补充："改得更专业一些"

### 场景 3：需要快速总结

**问题**：文档太长，需要提取要点

**解决**：
1. 打开 AI 助手
2. 点击 **总结** 快速操作
3. AI 会自动总结整个文档

### 场景 4：需要扩展内容

**问题**：内容太简略，需要更多细节

**解决**：
1. 选中需要扩展的段落
2. 点击 **扩展** 快速操作
3. AI 会增加更多细节和例子

### 场景 5：不理解某个概念

**问题**：文档中有不懂的概念

**解决**：
1. 选中概念名称
2. 点击 **解释** 快速操作
3. AI 会通俗易懂地解释

## 小技巧

### 技巧 1：明确你的需求

**不好的提问**：
```
帮我写点东西
```

**好的提问**：
```
帮我写一段 200 字左右的产品介绍，
重点突出创新性和用户价值
```

### 技巧 2：提供上下文

**不好的提问**：
```
这个怎么改？
```

**好的提问**：
```
这段话是写给技术人员看的，
帮我改得更专业、更准确
```

### 技巧 3：分步骤进行

**不好的做法**：
```
帮我写一篇完整的技术文章
```

**好的做法**：
```
1. 先生成文章大纲
2. 然后逐段扩展
3. 最后润色和优化
```

### 技巧 4：利用多轮对话

```
用户: 帮我写一段关于 AI 的介绍
AI: [生成内容]

用户: 太学术了，改得通俗一些
AI: [调整内容]

用户: 加上一些实际应用的例子
AI: [补充例子]
```

### 技巧 5：善用模板

不要从零开始，使用模板可以：
- 节省时间
- 确保结构完整
- 获得专业格式
- 激发灵感

## 注意事项

### ⚠️ 生成速度

- 流式输出需要时间
- 复杂内容生成较慢
- 可以随时点击 **停止** 按钮

### ⚠️ 内容准确性

- AI 生成的内容仅供参考
- 需要人工审核和修改
- 专业内容请咨询专家

### ⚠️ 隐私保护

- 不要输入敏感信息
- 不要输入个人隐私
- 不要输入商业机密

## 故障排除

### 问题：AI 没有响应

**可能原因**：
- 网络连接问题
- 服务器繁忙
- API 配置错误

**解决方法**：
1. 检查网络连接
2. 刷新页面重试
3. 联系管理员

### 问题：生成的内容不满意

**解决方法**：
1. 点击 **重新生成** 按钮
2. 提供更详细的要求
3. 使用多轮对话逐步优化

### 问题：浮动工具栏没有出现

**解决方法**：
1. 确保选中了文字
2. 检查是否在编辑器中
3. 刷新页面重试
