# AI 写作助手 - 设计说明

## 设计理念

### 参考对象

1. **Notion AI**
   - 斜杠命令触发
   - 浮动工具栏
   - 流畅的动画
   - 清晰的功能分类

2. **Grammarly**
   - 实时建议
   - 侧边栏展示
   - 卡片式建议
   - 一键应用

3. **飞书 AI**
   - 智能推荐
   - 多种写作模板
   - 对话式交互
   - 结果预览

### 核心原则

1. **非侵入式**：不干扰用户的主要工作流程
2. **即时响应**：快速反馈，流式输出
3. **易于发现**：明确的入口和操作提示
4. **灵活适配**：多种视图模式适应不同场景

## 视觉设计

### 配色方案

**主色调**：
- 渐变紫色：`from-purple-500 to-pink-500`
- 用于主要按钮、图标、强调元素

**辅助色**：
- 蓝色系：`blue-50` 到 `blue-700`
- 绿色系：`green-50` 到 `green-700`

- 主背景：`bg-white`

- 主文字：`text-purple-900`
- 次级文字：`text-purple-600`
- 辅助文字：`text-purple-500`

- 悬浮阴影：`shadow-lg`
- 强调阴影：`shadow-2xl`

### 动画效果

## 技术实现

### 流式输出

**客户端**：
```typescript
const reader = response.body?.getReader();
const decoder = new TextDecoder();

while (true) {
  const { done, value } = await reader.read();
  if (done) break;

  const chunk = decoder.decode(value, { stream: true });
  setContent(prev => prev + chunk);
}
```

**服务端**：
```typescript
const stream = new ReadableStream({
  async start(controller) {
    await streamAIResponse(messages, (chunk) => {
      controller.enqueue(encoder.encode(chunk));
    });
    controller.close();
  },
});
```

### 状态管理

使用 React Hooks：
- `useState`：本地状态
- `useEffect`：副作用处理
- `useRef`：DOM 引用和可变值
- `useCallback`：函数缓存

### 性能优化

1. **防抖输入**：
```typescript
const debouncedInput = useMemo(
  () => debounce(handleInput, 500),
  []
);
```

2. **虚拟滚动**：
- 长列表使用虚拟滚动
- 只渲染可见项
- 减少 DOM 节点

3. **懒加载**：
- 模板按需加载
- 图片懒加载
- 代码分割

4. **请求取消**：
```typescript
const abortController = new AbortController();
fetch(url, { signal: abortController.signal });
// 取消时
abortController.abort();
```
