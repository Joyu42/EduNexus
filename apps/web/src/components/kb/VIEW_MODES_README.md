# 知识宝库视图模式

## 概述

知识宝库现在支持 4 种不同的视图模式，提供更灵活的文档浏览体验。

## 视图模式

### 1. 列表视图 (List View)
- **默认视图**
- 传统的文档列表展示
- 支持搜索、筛选和排序
- 点击文档后在右侧编辑/预览

### 2. 卡片视图 (Card View)
- **网格布局**展示文档卡片
- 每个卡片显示：
  - 文档标题
  - 内容摘要（前 60-150 字）
  - 标签
  - 更新时间
- 支持 3 种卡片大小：小/中/大
- 悬停效果和点击动画
- 响应式网格布局

### 3. 时间轴视图 (Timeline View)
- **按时间线**展示文档
- 自动分组：
  - 今天
  - 昨天
  - 本周
  - 本月
  - 更早
- 时间线左侧显示日期标记
- 支持展开/折叠分组
- 显示文档摘要和标签

### 4. 看板视图 (Kanban View)
- **看板式**布局
- 默认 3 列：
  - 待学习
  - 学习中
  - 已完成
- 支持拖拽移动文档到不同列
- 每列显示文档数量
- 支持添加自定义列
- 拖拽改变文档状态（通过标签）

## 使用方法

### 切换视图
在知识宝库页面顶部工具栏，点击视图切换器按钮：
- 📋 列表视图
- 🔲 卡片视图
- 🕐 时间轴视图
- 📊 看板视图

### 视图记忆
系统会自动记住您选择的视图模式（存储在 localStorage），下次打开时自动恢复。

### 搜索和筛选
所有视图都支持：
- 全文搜索
- 标签筛选
- 日期范围筛选
- 逻辑模式（AND/OR）

## 技术实现

### 组件文件
- `view-switcher.tsx` - 视图切换器
- `card-view.tsx` - 卡片视图
- `timeline-view.tsx` - 时间轴视图
- `kanban-view.tsx` - 看板视图

### 动画
使用 Framer Motion 实现：
- 视图切换动画
- 卡片悬停效果
- 时间轴展开/折叠
- 看板拖拽反馈

### 状态管理
- 视图模式存储在组件状态
- 用户选择保存到 localStorage
- 文档状态通过标签管理

## 响应式设计

所有视图都针对不同屏幕尺寸优化：
- 桌面端：完整功能
- 平板：自适应布局
- 移动端：单列显示

## 快捷键

- `Ctrl+S` - 保存文档（列表视图）
- `Ctrl+E` - 切换编辑模式（列表视图）
- `Ctrl+Shift+N` - 快速创建笔记
- `Esc` - 退出全屏
