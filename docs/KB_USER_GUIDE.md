## 快速开始

### 创建第一个文档

1. 点击 "新建文档" 按钮或按 `Ctrl+N`
2. 输入文档标题
3. 开始编写内容
4. 按 `Ctrl+S` 保存

### 基本操作

- **新建文档**: `Ctrl+N`
- **保存文档**: `Ctrl+S`
- **搜索文档**: `Ctrl+F`
- **查看快捷键**: `Ctrl+/`

## 编辑器使用

### Tiptap 富文本编辑器

我们的编辑器支持丰富的格式和功能，让你的笔记更加生动。

#### 文本格式

- **粗体**: 选中文本后按 `Ctrl+B` 或点击工具栏的 **B** 按钮
- **斜体**: 选中文本后按 `Ctrl+I` 或点击工具栏的 *I* 按钮
- **删除线**: 使用工具栏的删除线按钮
- **行内代码**: 按 `Ctrl+E` 或使用工具栏

#### 标题

使用工具栏或快捷方式插入标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

#### 列表

- **无序列表**: 输入 `- ` 或 `* ` 开始
- **有序列表**: 输入 `1. ` 开始
- **任务列表**: 输入 `- [ ] ` 开始

示例：
```markdown
- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2

1. 第一步
2. 第二步
3. 第三步

- [ ] 待办事项 1
- [x] 已完成事项
- [ ] 待办事项 2
```

### 斜杠命令

按 `/` 键打开命令菜单，快速插入内容块：

| 命令 | 说明 | 快捷键 |
|------|------|--------|
| `/h1` | 一级标题 | - |
| `/h2` | 二级标题 | - |
| `/h3` | 三级标题 | - |
| `/ul` | 无序列表 | - |
| `/ol` | 有序列表 | - |
| `/task` | 任务列表 | - |
| `/code` | 代码块 | - |
| `/quote` | 引用 | - |
| `/table` | 表格 | - |
| `/hr` | 分隔线 | - |

**使用技巧**:
- 输入 `/` 后可以搜索命令
- 使用 `↑` `↓` 键导航
- 按 `Enter` 选择命令
- 按 `Esc` 关闭菜单

### 插入内容

#### 链接

1. 选中要添加链接的文本
2. 点击工具栏的链接按钮
3. 输入 URL
4. 确认

或使用 Markdown 语法：
```markdown
[链接文本](https://example.com)
```

#### 图片

1. 点击工具栏的图片按钮
2. 输入图片 URL
3. 确认

或使用 Markdown 语法：
```markdown
![图片描述](https://example.com/image.jpg)
```

#### 代码块

使用斜杠命令 `/code` 或工具栏按钮插入代码块：

\`\`\`javascript
function hello() {
  console.log("Hello, World!");
}
\`\`\`

支持语法高亮的语言包括：JavaScript、Python、Java、C++、Go、Rust 等。

#### 表格

使用斜杠命令 `/table` 插入 3x3 表格：

| 列1 | 列2 | 列3 |
| --- | --- | --- |
| 内容 | 内容 | 内容 |
| 内容 | 内容 | 内容 |

**表格操作**:
- 点击单元格编辑内容
- 拖动边框调整列宽
- 使用 `Tab` 键在单元格间导航

#### 引用

使用斜杠命令 `/quote` 或工具栏按钮：

> 这是一段引用文本
> 可以跨越多行

## 文档管理

### 标签系统

标签帮助你组织和分类文档。

#### 添加标签

1. 在文档编辑界面找到标签输入框
2. 输入标签名称
3. 按 `Enter` 添加
4. 可以添加多个标签

#### 使用标签

- **查看所有标签**: 打开文档侧边栏的"标签"标签页
- **按标签筛选**: 点击标签查看该标签下的所有文档
- **热门标签**: 标签按使用次数排序，常用标签显示在前面

**标签最佳实践**:
- 使用简短、描述性的标签名
- 建立一致的标签体系（如：`项目-名称`、`类型-内容`）
- 不要创建过多标签，保持标签数量在 20-30 个
- 定期清理不再使用的标签

### 收藏功能

收藏重要文档以便快速访问。

#### 添加收藏

1. 打开文档
2. 点击星标按钮
3. 文档添加到收藏列表

#### 访问收藏

1. 打开文档侧边栏
2. 切换到"收藏"标签页
3. 点击文档快速打开

### 最近访问

系统自动记录你最近访问的文档。

#### 查看最近访问

1. 打开文档侧边栏
2. 切换到"最近"标签页
3. 查看最近访问的 10 个文档

#### 管理历史

- **清除单条**: 悬停在文档上，点击 `×` 按钮
- **清除全部**: 暂不支持（保留最近 50 条）

## 搜索功能

### 基本搜索

1. 按 `Ctrl+F` 或点击搜索框
2. 输入搜索关键词
3. 查看搜索结果

搜索范围包括：
- 文档标题
- 文档内容
- 文档标签

### 高级搜索

#### 标签筛选

1. 在搜索界面点击"筛选"
2. 选择一个或多个标签
3. 只显示包含这些标签的文档

#### 日期范围

1. 在搜索界面点击"筛选"
2. 设置开始日期和结束日期
3. 只显示该时间范围内更新的文档

#### 排序选项

- **相关性**: 按搜索匹配度排序（默认）
- **日期**: 按更新时间排序
- **标题**: 按标题字母顺序排序

每种排序都支持升序和降序。

### 搜索历史

系统自动保存你的搜索历史。

#### 使用搜索历史

1. 点击搜索框（不输入内容）
2. 查看最近的搜索历史
3. 点击历史记录快速重复搜索

#### 管理历史

- **清除全部**: 点击"清除"按钮
- **删除单条**: 悬停在历史记录上，点击 `×` 按钮

### 搜索建议

输入搜索关键词时，系统会提供实时建议：

- 匹配的文档标题
- 匹配的标签
- 相关的搜索词

使用 `↑` `↓` 键导航建议，按 `Enter` 选择。

## 导出和分享

### 导出单个文档

#### 导出为 Markdown

1. 打开文档
2. 点击"导出"按钮
3. 选择"Markdown"
4. 文件自动下载

导出的 Markdown 文件包含：
- 文档标题
- 文档内容
- 标签
- 创建和更新时间

#### 导出为 HTML

1. 打开文档
2. 点击"导出"按钮
3. 选择"HTML"
4. 文件自动下载

HTML 文件包含完整的样式，可以直接在浏览器中查看。

#### 导出为 PDF

1. 打开文档
2. 点击"导出"按钮
3. 选择"PDF"
4. 浏览器打开打印预览
5. 选择"保存为 PDF"

### 批量导出

1. 在文档列表中选择多个文档
2. 点击"批量导出"
3. 选择导出格式
4. 文件逐个下载

### 版本历史

系统自动保存文档的版本历史。

#### 查看版本历史

1. 打开文档
2. 点击"版本历史"按钮
3. 查看所有历史版本

#### 比较版本

1. 在版本历史中选择两个版本
2. 点击"比较"
3. 查看差异

差异显示：
- `- 红色`: 删除的内容
- `+ 绿色`: 添加的内容
- `  灰色`: 未改变的内容

#### 恢复版本

1. 在版本历史中选择要恢复的版本
2. 点击"恢复"
3. 确认恢复

**注意**: 恢复版本会创建一个新版本，不会删除历史记录。

## 最佳实践

### 文档组织

1. **使用清晰的标题**
   - 标题应该简洁明了
   - 使用层级结构（H1 > H2 > H3）
   - 避免过长的标题

2. **合理使用标签**
   - 每个文档 3-5 个标签
   - 建立一致的标签体系
   - 定期清理不用的标签

3. **定期整理**
   - 每周回顾文档
   - 删除过时内容
   - 更新重要文档

### 内容编写

1. **使用 Markdown 格式**
   - 保持格式一致
   - 使用列表和表格
   - 添加代码块和引用

2. **添加元数据**
   - 使用标签分类
   - 添加创建日期
   - 记录更新原因

3. **保持简洁**
   - 一个文档一个主题
   - 使用简短的段落
   - 避免冗余内容

### 搜索技巧

1. **使用精确关键词**
   - 使用文档中的实际词语
   - 避免过于宽泛的搜索
   - 尝试不同的关键词

2. **利用筛选功能**
   - 按标签缩小范围
   - 使用日期范围
   - 组合多个筛选条件

3. **查看搜索历史**
   - 重复常用搜索
   - 优化搜索词
   - 保存有用的搜索

### 备份和导出

1. **定期备份**
   - 每周导出重要文档
   - 使用多种格式
   - 保存到云存储

2. **版本管理**
   - 重要修改前查看历史
   - 定期清理旧版本
   - 记录变更说明

3. **分享文档**
   - 导出为 HTML 分享
   - 使用 PDF 打印
   - Markdown 用于协作

### 性能优化

1. **控制文档大小**
   - 单个文档不超过 10MB
   - 大文档拆分为多个
   - 压缩图片

2. **定期清理**
   - 删除不用的文档
   - 清理搜索历史
   - 清理旧版本

3. **使用标签而非文件夹**
   - 标签更灵活
   - 支持多分类
   - 便于搜索

## 常见问题

### Q: 如何批量添加标签？
A: 目前需要逐个文档添加标签。建议在创建文档时就添加标签。

### Q: 搜索不到文档怎么办？
A: 检查搜索关键词是否正确，尝试使用标签筛选，或查看最近访问列表。

### Q: 如何恢复误删的文档？
A: 目前没有回收站功能。建议定期导出重要文档作为备份。

### Q: 版本历史保存多久？
A: 默认保留最近 10 个版本。可以手动清理旧版本。

### Q: 支持协作编辑吗？
A: 目前不支持实时协作。可以导出文档分享给他人。

### Q: 如何更改主题？
A: 点击右上角的主题切换按钮，选择浅色、深色或跟随系统。
