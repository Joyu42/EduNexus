## 🧪 功能测试步骤

### 测试 1: 自动保存
1. 访问 http://localhost:3003/kb
2. 创建或打开一个笔记
3. 编辑内容
4. 观察右上角保存状态指示器
5. 等待2秒后应显示 "已保存"

### 测试 2: 快速创建 - 快捷键
1. 在知识宝库页面按 Ctrl+Shift+N (或 Cmd+Shift+N)
2. 应弹出快速创建对话框
3. 选择创建类型并填写内容
4. 确认创建成功

### 测试 3: 快速创建 - 浮动按钮
1. 查看页面右下角的浮动按钮
2. 点击浮动按钮
3. 应弹出快速创建对话框
4. 测试不同的创建类型

### 测试 4: 从选中文本创建
1. 在笔记中选中一段文本
2. 右键点击选中的文本
3. 选择"从选中文本创建笔记"
4. 验证新笔记包含选中的内容

### 测试 5: 离线保存
1. 打开浏览器开发者工具
2. 切换到 Network 标签
3. 勾选 "Offline" 模拟离线状态
4. 编辑笔记内容
5. 观察保存状态（应显示离线保存）
6. 取消 "Offline"，恢复在线
7. 验证内容自动同步到服务器

### 测试 6: 数据同步
1. 创建包含标签的笔记 (例如: #JavaScript #学习)
2. 创建包含双链的笔记 (例如: [[React]] 是一个框架)
3. 访问知识星图页面
4. 验证新节点和关系已自动创建
5. 访问成长地图页面
6. 验证学习活动已记录

### 测试 7: 内容提取
1. 创建包含以下内容的笔记：
   ```markdown
   # 测试笔记

   这是一个测试 #测试 #自动提取

   参考文档：[[知识图谱]] 和 [[学习路径]]

   ```javascript
   console.log('Hello World');
   ```

   数学公式：$$E = mc^2$$
   ```
2. 保存后检查标签是否自动提取
3. 验证双链引用是否识别
4. 确认代码块和公式被正确解析

## 📝 注意事项

1. 开发服务器运行在 http://localhost:3003
2. 自动保存延迟为 2 秒，可在 `use-auto-save.ts` 中配置
3. 离线保存使用 IndexedDB，需要浏览器支持
4. 数据同步采用批量处理，每批最多 5 个任务
5. 快捷键在不同操作系统上有所不同（Ctrl/Cmd）
