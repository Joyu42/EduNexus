## 📊 技术栈

- **编辑器**: Tiptap + ProseMirror
- **语法高亮**: Lowlight
- **存储**: IndexedDB
- **UI**: Radix UI + Tailwind CSS
- **日期处理**: date-fns
- **类型安全**: TypeScript

## 🎨 设计原则

1. **简洁直观**: 界面简洁，操作直观
2. **性能优先**: 快速响应，流畅体验
3. **渐进增强**: 基础功能稳定，高级功能可选
4. **可访问性**: 支持键盘导航，屏幕阅读器友好
5. **响应式**: 适配各种屏幕尺寸

## 💡 使用提示

1. 使用 `/` 快速插入内容块
2. 使用 `Ctrl+/` 查看所有快捷键
3. 使用标签组织文档
4. 收藏重要文档以便快速访问
5. 定期导出重要文档作为备份
6. 使用版本历史追踪文档变更
