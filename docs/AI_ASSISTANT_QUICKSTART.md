# 全局 AI 助手 - 快速启动指南

## 🚀 快速开始

### 1. 打开助手
按 `Cmd/Ctrl + K` 或点击右下角的紫色按钮

### 2. 开始对话
- 直接输入问题
- 或点击快速操作按钮

### 3. 管理对话
- 复制回复：点击消息下方的复制图标
- 清空对话：点击"清空对话"按钮
- 重新开始：点击头部的刷新图标

## 📍 功能特性

### 上下文感知
助手会根据当前页面自动切换模式：

| 页面 | 模式 | 功能 |
|------|------|------|
| `/kb` | 写作助手 | 扩展内容、总结要点、优化表达 |
| `/workspace` | 学习助手 | 解释概念、举例说明、对比分析 |
| `/practice` | 答题助手 | 解题提示、思路引导、答案检查 |
| 其他 | 通用助手 | 问答、搜索、建议 |

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + K` | 打开/关闭助手 |
| `Enter` | 发送消息 |
| `Shift + Enter` | 换行 |

### 窗口操作
- **拖拽**：按住头部区域拖动
- **最小化**：点击 `-` 按钮
- **关闭**：点击 `×` 按钮
- **重置**：点击刷新图标

## 🎯 使用场景

### 场景 1：写作辅助（知识库页面）
```
1. 在知识库编辑笔记
2. 按 Cmd/Ctrl + K 打开助手
3. 选择"扩展内容"
4. 输入需要扩展的段落
5. 复制 AI 的建议到笔记中
```

### 场景 2：学习答疑（工作区页面）
```
1. 在工作区学习时遇到问题
2. 按 Cmd/Ctrl + K 打开助手
3. 直接提问："什么是递归？"
4. 查看 AI 的详细解释
5. 继续追问深入理解
```

### 场景 3：练习辅导（练习页面）
```
1. 做练习题遇到困难
2. 按 Cmd/Ctrl + K 打开助手
3. 选择"给个提示"
4. 描述题目
5. 根据提示继续思考
```
