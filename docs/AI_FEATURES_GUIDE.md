# AI 智能功能使用指南

## 快速开始

### 1. 配置环境变量

在 `.env.local` 文件中添加：

```env
MODELSCOPE_API_KEY=your_api_key_here
MODELSCOPE_BASE_URL=https://api-inference.modelscope.cn/v1
MODELSCOPE_CHAT_MODEL=Qwen/Qwen3.5-122B-A10B
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 访问功能

- **知识库 AI 功能**: http://localhost:3000/kb
- **学习工作区**: http://localhost:3000/workspace

## 功能详解

### 📝 文档摘要

**位置**: 知识库 → 选择文档 → 右侧面板 → "摘要" 标签页

**功能**:
- 自动生成 200-300 字的文档摘要
- 提取 3-5 个关键要点
- 生成文档大纲结构
- 显示字数和预计阅读时间

**使用步骤**:
1. 在知识库中选择一个文档
2. 点击右侧面板的"摘要"标签
3. 点击"生成摘要"按钮
4. 等待 AI 分析完成
5. 可以复制或导出摘要

**提示**:
- 摘要会保留在当前会话中
- 可以点击"重新生成"获取不同的摘要
- 支持导出为 Markdown 格式

### 🏷️ 关键词提取

**位置**: 知识库 → 选择文档 → 右侧面板 → "关键词" 标签页

**功能**:
- 提取 5-10 个关键词
- 按重要性排序（字体大小表示重要性）
- 生成建议标签
- 一键添加标签到文档

**使用步骤**:
1. 在知识库中选择一个文档
2. 点击右侧面板的"关键词"标签
3. 点击"提取关键词"按钮
4. 查看关键词云和建议标签
5. 点击标签添加到文档

**提示**:
- 关键词云中字体越大表示重要性越高
- 已添加的标签会显示为绿色
- 可以点击"全部添加"一次性添加所有建议标签

### 🗺️ 思维导图

**位置**: 知识库 → 选择文档 → 右侧面板 → "导图" 标签页

**功能**:
- 分析文档结构生成思维导图
- 交互式可视化展示
- 支持缩放、拖拽
- 支持全屏查看

**使用步骤**:
1. 在知识库中选择一个文档
2. 点击右侧面板的"导图"标签
3. 点击"生成思维导图"按钮
4. 等待生成完成
5. 使用鼠标拖拽和滚轮缩放

**提示**:
- 橙色节点表示根节点（文档标题）
- 黄色节点表示分支节点
- 浅黄色节点表示叶子节点
- 点击全屏按钮可以获得更好的查看体验

### 📚 学习计划

**位置**: 学习工作区 → 右侧面板 → "计划" 标签页

**功能**:
- 基于知识库内容生成学习计划
- 分析知识点覆盖度
- 推荐学习顺序
- 估算学习时间
- 生成具体学习任务

**使用步骤**:
1. 打开学习工作区
2. 点击右侧面板的"计划"标签
3. （可选）输入学习目标
4. 点击"生成学习计划"按钮
5. 查看生成的学习计划
6. 可以导出为 Markdown 文件

**提示**:
- 学习计划会根据知识库中的文档数量和内容生成
- 建议在知识库中至少有 5 个文档后再生成学习计划
- 可以多次生成以获得不同的学习路径

### 💬 知识库问答

**位置**: 学习工作区 → 右侧面板 → "问答" 标签页

**功能**:
- 基于知识库内容回答问题
- 引用相关文档片段
- 支持追问和深入讨论
- 保留对话历史

**使用步骤**:
1. 打开学习工作区
2. 点击右侧面板的"问答"标签
3. 在输入框中输入问题
4. 按 Enter 发送（Shift+Enter 换行）
5. 查看 AI 的回答

**提示**:
- AI 会优先使用知识库中的信息回答
- 如果知识库中没有相关信息，AI 会明确说明
- 可以点击垃圾桶图标清除对话历史
- 右上角显示当前知识库的文档数量

## 常见问题

### Q: 为什么生成速度很慢？

A: AI 生成需要调用 ModelScope API，速度取决于：
- 网络连接质量
- API 服务器负载
- 文档长度和复杂度

通常需要 5-15 秒。

### Q: 生成失败怎么办？

A: 检查以下几点：
1. 确认 MODELSCOPE_API_KEY 已正确配置
2. 检查网络连接
3. 查看浏览器控制台的错误信息
4. 尝试重新生成

### Q: 可以自定义 AI 模型吗？

A: 可以通过修改环境变量 `MODELSCOPE_CHAT_MODEL` 来更换模型。
推荐使用 Qwen 系列模型。

### Q: 生成的内容可以编辑吗？

A: 目前生成的内容不能直接编辑，但可以：
- 复制到剪贴板后编辑
- 导出为文件后编辑
- 重新生成获取不同的结果

### Q: 支持哪些语言？

A: 目前主要支持中文和英文。AI 会根据文档内容自动选择语言。

## 性能优化建议

1. **文档长度**: 建议单个文档不超过 5000 字，过长的文档可能导致生成时间过长
2. **批量处理**: 如果需要处理多个文档，建议逐个处理而不是同时处理
3. **缓存**: 生成的结果会在当前会话中保留，切换文档后会清除
4. **网络**: 使用稳定的网络连接以获得最佳体验
