# 全局 AI 悬浮助手使用文档

## 概述

全局 AI 悬浮助手是一个智能的、上下文感知的对话助手，可以在 EduNexus 平台的任何页面使用。它会根据当前页面自动调整功能和提示，为用户提供最合适的帮助。

## 核心特性

### 1. 全局可用
- **快捷键唤起**：按 `Cmd/Ctrl + K` 在任何页面快速打开/关闭
- **悬浮设计**：浮动在页面右下角，不影响正常操作
- **可拖拽**：点击头部区域可拖拽到任意位置
- **位置记忆**：自动记住用户的位置偏好

### 2. 上下文感知

助手会根据当前页面自动切换模式：

#### 知识库页面 - 写作助手
- **功能**：帮助完善笔记、优化表达、补充内容
- **快速操作**：
  - 📝 扩展内容：添加更多细节和例子
  - 📋 总结要点：提取核心要点
  - ✨ 优化表达：改善文字表达
  - 🗂️ 整理结构：优化内容结构

#### 工作区页面 - 学习助手
- **功能**：解答疑问、解释概念、提供学习建议
- **快速操作**：
  - 💡 解释概念：详细解释知识点
  - 📚 举例说明：提供实际例子
  - ⚖️ 对比分析：对比不同概念
  - 🎯 练习建议：提供练习方法

#### 练习页面 - 答题助手
- **功能**：引导思考、提供解题思路（不直接给答案）
- **快速操作**：
  - 💭 给个提示：提供解题提示
  - 🧭 解题思路：引导思考方向
  - ✅ 检查答案：验证答案思路
  - 🔄 类似题目：推荐相似练习

#### 其他页面 - 通用助手
- **功能**：通用问答和帮助
- **快速操作**：
  - ❓ 提问：询问问题
  - 🔍 搜索：搜索内容
  - 💡 建议：获取建议
  - 🆘 帮助：寻求帮助

### 3. 交互功能

#### 对话管理
- **流式输出**：实时显示 AI 回复
- **历史记录**：保留完整对话历史
- **清空对话**：一键清空当前对话
- **重新开始**：重置所有状态

#### 消息操作
- **复制消息**：点击复制按钮复制 AI 回复
- **快速输入**：点击快速操作按钮自动填充提示词
- **键盘快捷键**：
  - `Enter`：发送消息
  - `Shift + Enter`：换行
  - `Cmd/Ctrl + K`：打开/关闭助手

#### 窗口控制
- **最小化**：点击 `-` 按钮最小化窗口
- **关闭**：点击 `×` 按钮关闭助手
- **拖拽**：按住头部区域拖拽到任意位置

## 使用场景

### 场景 1：写作辅助
```
用户在知识库编辑笔记时：
1. 按 Cmd/Ctrl + K 打开助手
2. 选择"扩展内容"快速操作
3. 输入需要扩展的内容
4. AI 提供详细的扩展建议
5. 点击复制按钮，粘贴到笔记中
```

### 场景 2：学习答疑
```
用户在工作区学习时遇到问题：
1. 按 Cmd/Ctrl + K 打开助手
2. 直接输入问题："什么是递归？"
3. AI 提供详细解释和例子
4. 继续追问："能给我一个实际的代码例子吗？"
5. AI 提供代码示例
```

### 场景 3：练习辅导
```
用户在做练习题时遇到困难：
1. 按 Cmd/Ctrl + K 打开助手
2. 选择"给个提示"快速操作
3. 描述题目内容
4. AI 提供解题提示（不直接给答案）
5. 根据提示继续思考
```

## 技术实现

### 组件结构
```
apps/web/src/
├── components/
│   └── global/
│       └── global-ai-assistant.tsx    # 主组件
├── lib/
│   ├── ai/
│   │   └── context-adapter.ts         # 上下文适配器
│   └── hooks/
│       ├── use-keyboard-shortcut.ts   # 快捷键 Hook
│       └── use-draggable.ts           # 拖拽 Hook
```

### 核心技术
- **React Hooks**：状态管理和副作用处理
- **Framer Motion**：流畅的动画效果
- **localStorage**：位置记忆
- **Next.js App Router**：路由检测
- **Tailwind CSS**：样式设计

### API 集成
助手使用现有的 Agent API：
```typescript
POST /api/workspace/agent/chat
{
  message: string,
  history: Array<{role, content}>,
  config: {
    systemPrompt: string
  }
}
```

## 自定义配置

### 修改快捷键
编辑 `global-ai-assistant.tsx`：
```typescript
useKeyboardShortcut(
  () => setIsOpen(prev => !prev),
  { key: 'k', ctrl: true }  // 修改为其他按键
);
```

### 添加新的上下文模式
编辑 `context-adapter.ts`：
```typescript
export function getAIContext(pathname: string): AIContext {
  // 添加新的路由匹配
  if (pathname.startsWith('/your-page')) {
    return {
      mode: 'custom',
      title: '自定义助手',
      placeholder: '自定义提示',
      systemPrompt: '自定义系统提示',
      quickActions: [
        // 自定义快速操作
      ],
    };
  }
  // ...
}
```

### 调整样式
组件使用 Tailwind CSS，可以直接修改类名：
```typescript
// 修改主题色
className="bg-gradient-to-br from-purple-500 to-pink-500"
// 改为其他颜色
className="bg-gradient-to-br from-blue-500 to-cyan-500"
```

## 最佳实践

## 常见问题

### Q: 助手没有响应？
A: 检查网络连接，确保 API 服务正常运行。

### Q: 如何更改助手位置？
A: 点击头部区域拖拽到任意位置，位置会自动保存。

### Q: 快捷键不生效？
A: 确保没有其他应用占用相同快捷键，尝试刷新页面。

### Q: 如何清除保存的位置？
A: 打开浏览器开发者工具，在 Console 中执行：
```javascript
localStorage.removeItem('ai-assistant-position');
```

### Q: 助手回复不准确？
A: 尝试提供更多上下文信息，或使用快速操作引导 AI。
