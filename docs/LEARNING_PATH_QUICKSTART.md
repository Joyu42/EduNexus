# 学习路径系统 - 快速开始

## 快速体验

### 1. 访问学习路径

在侧边栏点击 **🛤️ 学习路径** 或访问 `/learning-paths`

### 2. 浏览路径市场

点击 **"浏览市场"** 按钮，查看预设的学习路径模板：

- **前端开发入门**：HTML → CSS → JavaScript 完整路径
- **Python 编程入门**：Python 基础语法和实践
- **数据结构与算法**：系统学习常用数据结构

### 3. 开始学习

1. 选择一个感兴趣的模板
2. 点击 **"开始学习"** 按钮
3. 在可视化界面中：
   - 点击节点查看学习内容
   - 完成后点击 **"标记为完成"**
   - 查看右侧的学习统计
4. 完成所有节点后获得证书

### 4. 创建自定义路径

1. 点击 **"创建路径"** 按钮
2. 在编辑器中：
   - 点击 **"添加节点"** 添加学习内容
   - 拖拽节点调整位置
   - 连接节点设置学习顺序
   - 点击 **"设置"** 配置路径信息
3. 点击 **"保存"** 保存你的路径
4. 点击 **"预览"** 查看效果

## 核心概念

### 节点类型

- **📄 文档**：学习文档、教程、文章
- **🎥 视频**：视频教程、讲座
- **💻 练习**：编程练习、实战项目
- **✅ 测验**：知识测验、考试

### 难度级别

- **初级**：适合零基础学习者
- **中级**：需要一定基础知识
- **高级**：适合有经验的学习者

### 进度追踪

系统自动追踪你的学习进度：
- 已完成节点数
- 总体完成百分比
- 学习开始和完成时间
- 当前学习节点

## 使用技巧

### 编辑器技巧

1. **快速布局**：使用 React Flow 的自动布局功能
2. **批量操作**：按住 Shift 选择多个节点
3. **缩放视图**：使用鼠标滚轮或右下角控制器
4. **小地图**：右下角小地图快速导航

### 学习技巧

1. **按顺序学习**：遵循节点连接的依赖关系
2. **记录笔记**：在节点详情中添加学习笔记
3. **定期复习**：查看已完成的节点进行复习
4. **设定目标**：根据预计时长制定学习计划

### 导入导出

**分享路径：**
1. 在编辑器中打开路径
2. 点击 **"导出"** 下载 JSON 文件
3. 分享文件给其他人

**导入路径：**
1. 点击 **"导入"** 按钮
2. 选择 JSON 文件
3. 路径自动加载

## 常见问题

### Q: 数据保存在哪里？
A: 数据保存在浏览器的 IndexedDB 中，仅在本地存储。

### Q: 可以在多个设备间同步吗？
A: 当前版本不支持云同步，可以通过导出/导入功能手动同步。

### Q: 如何删除路径？
A: 在"我的学习路径"页面，点击路径卡片上的删除按钮。

### Q: 证书有效吗？
A: 当前证书仅作为学习记录，不具备官方认证效力。

### Q: 可以修改预设模板吗？
A: 可以！点击下载图标克隆模板，然后在编辑器中自由修改。
